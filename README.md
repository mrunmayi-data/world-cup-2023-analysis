# ICC Men's Cricket World Cup 2023 Analysis

## Project Overview
This project analyzes ICC Men's Cricket World Cup 2023 data using Python and Data Visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Plotly
- Excel
- MySQL

## Key Insights
- Daryl Mitchell was the top scorer multiple times.
- Adam Zampa delivered the best bowling performances.
- Wankhede Stadium favored batting first.
- Travis Head and Mohammad Shami won most Player of the Match awards.

## Conclusion
The project provides insights into player performances, stadium behavior, and match-winning patterns during the ICC Cricket World Cup 2023.
